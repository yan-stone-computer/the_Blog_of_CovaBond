什么是密码哈希？

密码哈希是将用户密码转换成一串不可逆的字符串的过程。即使数据库被泄露，攻击者也无法直接获取用户的原始密码。这是一个至关重要的安全措施，因为用户经常在多个网站使用相同的密码。

与加密不同，哈希是单向的——你无法从哈希值还原出原始密码。这使得哈希成为存储密码的理想选择。

重要提示

永远不要存储用户的明文密码！这是安全领域的基本原则。

为什么需要加盐？

问题：如果两个用户使用相同的密码，他们的哈希值也会相同。攻击者可以使用"彩虹表"攻击——预先计算常见密码的哈希值，然后与数据库中的哈希值比对。

解决：为每个密码添加唯一的随机值（盐），即使密码相同，哈希值也不同。

# 不加盐的情况
hash("password123") = "ef92b778bafe771e89245b89ecbc08a44a4e166c06659911881f383d44719133"

# 加盐后的情况
hash("password123" + "salt1") = "a1b2c3d4..."
hash("password123" + "salt2") = "x9y8z7w6..."

为什么需要慢哈希？

问题：快速哈希算法（如 MD5、SHA-1、SHA-256）让攻击者可以快速尝试大量密码。现代 GPU 每秒可以计算数十亿次 SHA-256 哈希。

解决：使用计算密集型的慢哈希算法，增加攻击者的成本。理想情况下，单次哈希计算应该需要几百毫秒。

警告

不要使用 MD5、SHA-1 或 SHA-256 来存储密码！它们设计用于快速计算，不适合密码存储。

Argon2 算法简介

Argon2 是目前最推荐的密码哈希算法，它是 2015 年 Password Hashing Competition（PHC）的获胜者。它有三个变体：

  • Argon2d：抵抗 GPU 攻击，但不抵抗侧信道攻击
  • Argon2i：抵抗侧信道攻击，但较慢
  • Argon2id：混合版本，推荐使用

Argon2 的主要参数：

  • time_cost：迭代次数，增加计算时间
  • memory_cost：内存使用量，增加 GPU 攻击难度
  • parallelism：并行线程数

Python 实现

使用 argon2-cffi 库实现安全的密码哈希：

from argon2 import PasswordHasher
from argon2.exceptions import VerifyMismatchError

class PasswordManager:
    def __init__(self):
        self.ph = PasswordHasher(
            time_cost=3,        # 迭代次数
            memory_cost=65536,  # 64MB 内存
            parallelism=4,      # 4 个线程
            hash_len=32,        # 哈希长度
            salt_len=16         # 盐长度
        )
    
    def hash_password(self, password: str) -> str:
        """生成密码哈希"""
        return self.ph.hash(password)
    
    def verify_password(self, hash_str: str, password: str) -> bool:
        """验证密码"""
        try:
            self.ph.verify(hash_str, password)
            return True
        except VerifyMismatchError:
            return False

# 使用示例
pm = PasswordManager()

# 注册时存储密码
password = "user_password_123"
hashed = pm.hash_password(password)
print(f"存储的哈希: {hashed}")

# 登录时验证密码
is_valid = pm.verify_password(hashed, password)
print(f"密码正确: {is_valid}")

C++ Qt 实现

在 Qt 中，我们可以使用迭代式 SHA-256 来实现慢哈希：

#include <QCryptographicHash>
#include <QRandomGenerator>

class PasswordManager {
public:
    static const int SALT_LENGTH = 16;
    static const int ITERATIONS = 100000;
    
    static QByteArray generateSalt() {
        QByteArray salt;
        salt.resize(SALT_LENGTH);
        QRandomGenerator::system()->fillRange(
            reinterpret_cast<quint32*>(salt.data()), 
            SALT_LENGTH / 4
        );
        return salt;
    }
    
    static QByteArray hashPassword(const QString& password, 
                                    const QByteArray& salt) {
        QByteArray data = password.toUtf8() + salt;
        
        for (int i = 0; i < ITERATIONS; ++i) {
            data = QCryptographicHash::hash(
                data, 
                QCryptographicHash::Sha256
            );
        }
        
        return data.toHex();
    }
    
    static bool verifyPassword(const QString& password,
                               const QByteArray& salt,
                               const QByteArray& hash) {
        return hashPassword(password, salt) == hash;
    }
};

// 使用示例
QString password = "user_password_123";
QByteArray salt = PasswordManager::generateSalt();
QByteArray hash = PasswordManager::hashPassword(password, salt);

// 存储 salt 和 hash 到数据库
// 验证时从数据库读取 salt 和 hash
bool valid = PasswordManager::verifyPassword(password, salt, hash);
