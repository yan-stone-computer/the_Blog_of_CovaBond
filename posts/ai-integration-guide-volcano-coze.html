AI大模型集成指南：火山引擎与Coze Bot实战

火山引擎 Coze Bot REST API RAG AI对话

在天枢便签、智能垃圾分类、竞语星河等项目中，我深度集成了火山引擎大模型和Coze Bot AI能力。本文将分享AI大模型集成的实战经验，包括API调用、知识库构建、对话流设计等核心技术。

AI集成方案选型

在选择AI服务提供商时，我主要考虑了以下几个维度：

  • 模型能力：文本生成、图像理解、多轮对话等
  • 响应速度：首token延迟、流式输出支持
  • 成本控制：token定价、免费额度
  • 易用性：SDK支持、文档完善度

最终选择了火山引擎作为主力大模型服务，Coze Bot作为对话工作流编排平台。

火山引擎大模型集成

1. API认证与初始化

import requests
import json

class VolcanoEngineClient:
    def __init__(self, api_key, model="doubao-pro-32k"):
        self.api_key = api_key
        self.model = model
        self.base_url = "https://ark.cn-beijing.volces.com/api/v3"
        
    def chat_completion(self, messages, stream=False):
        headers = {
            "Authorization": f"Bearer {self.api_key}",
            "Content-Type": "application/json"
        }
        
        data = {
            "model": self.model,
            "messages": messages,
            "stream": stream,
            "temperature": 0.7,
            "max_tokens": 2000
        }
        
        response = requests.post(
            f"{self.base_url}/chat/completions",
            headers=headers,
            json=data,
            stream=stream
        )
        
        return response

2. 流式输出实现

为了提供更好的用户体验，AI回复采用流式输出，逐字显示：

def stream_chat(self, messages, callback):
    """流式对话，callback接收每个token"""
    response = self.chat_completion(messages, stream=True)
    
    full_response = ""
    for line in response.iter_lines():
        if line:
            line = line.decode('utf-8')
            if line.startswith('data: '):
                data = line[6:]
                if data == '[DONE]':
                    break
                try:
                    chunk = json.loads(data)
                    delta = chunk['choices'][0]['delta']
                    if 'content' in delta:
                        content = delta['content']
                        full_response += content
                        callback(content)  # 实时更新UI
                except:
                    pass
    
    return full_response

3. 带上下文的对话

对话上下文管理策略：
  • 维护最近N轮对话历史（通常5-10轮）
  • 计算token数量，超出限制时裁剪最早的消息
  • 系统提示词(system prompt)始终保留
class ConversationManager:
    def __init__(self, max_history=10):
        self.messages = []
        self.max_history = max_history
        
    def set_system_prompt(self, prompt):
        """设置系统提示词"""
        self.messages = [{"role": "system", "content": prompt}]
        
    def add_user_message(self, content):
        """添加用户消息"""
        self.messages.append({"role": "user", "content": content})
        self._trim_history()
        
    def add_assistant_message(self, content):
        """添加助手回复"""
        self.messages.append({"role": "assistant", "content": content})
        
    def _trim_history(self):
        """裁剪历史记录，保留系统提示和最近N轮"""
        if len(self.messages) > self.max_history * 2 + 1:
            # 保留system + 最近N轮
            self.messages = [self.messages[0]] + \
                          self.messages[-(self.max_history * 2):]
    
    def get_messages(self):
        return self.messages

Coze API集成

class CozeClient:
    def __init__(self, api_key, bot_id):
        self.api_key = api_key
        self.bot_id = bot_id
        self.base_url = "https://api.coze.cn"
        
    def chat(self, query, conversation_id=None, user_id="default"):
        """调用Coze Bot进行对话"""
        headers = {
            "Authorization": f"Bearer {self.api_key}",
            "Content-Type": "application/json"
        }
        
        data = {
            "bot_id": self.bot_id,
            "user": user_id,
            "query": query,
            "stream": False
        }
        
        if conversation_id:
            data["conversation_id"] = conversation_id
            
        response = requests.post(
            f"{self.base_url}/open_api/v2/chat",
            headers=headers,
            json=data
        )
        
        result = response.json()
        return {
            "reply": result.get("messages", [{}])[0].get("content", ""),
            "conversation_id": result.get("conversation_id")
        }

RAG知识库构建

在天枢便签中，我实现了基于RAG(Retrieval-Augmented Generation)的知识库问答功能：

知识库架构

RAG流程：
  1. 文档切分：将长文档切分为适当长度的chunk
  2. 向量化：使用embedding模型将文本转为向量
  3. 向量存储：存入向量数据库(如Milvus、Pinecone)
  4. 语义检索：根据查询向量检索相关文档
  5. 增强生成：将检索结果作为上下文输入大模型

向量检索实现

class KnowledgeBase:
    def __init__(self, api_key):
        self.api_key = api_key
        self.documents = []  # 简化示例，实际使用向量数据库
        self.embeddings = []
        
    def add_document(self, text, metadata=None):
        """添加文档到知识库"""
        embedding = self._get_embedding(text)
        self.documents.append({
            "text": text,
            "metadata": metadata
        })
        self.embeddings.append(embedding)
        
    def _get_embedding(self, text):
        """获取文本向量表示"""
        headers = {
            "Authorization": f"Bearer {self.api_key}",
            "Content-Type": "application/json"
        }
        
        response = requests.post(
            "https://ark.cn-beijing.volces.com/api/v3/embeddings",
            headers=headers,
            json={"model": "doubao-embedding", "input": text}
        )
        
        return response.json()["data"][0]["embedding"]
    
    def search(self, query, top_k=3):
        """语义搜索相关文档"""
        query_embedding = self._get_embedding(query)
        
        # 计算余弦相似度
        similarities = []
        for doc_embedding in self.embeddings:
            sim = self._cosine_similarity(query_embedding, doc_embedding)
            similarities.append(sim)
            
        # 返回最相关的文档
        top_indices = np.argsort(similarities)[-top_k:][::-1]
        return [self.documents[i] for i in top_indices]
    
    def _cosine_similarity(self, a, b):
        return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

AI图像识别实战

智能垃圾分类项目的核心功能是AI图像识别，以下是完整实现：

class ImageRecognitionService:
    def __init__(self, api_key):
        self.api_key = api_key
        
    def recognize(self, image_path):
        """识别图片中的垃圾类别"""
        # 1. 图片预处理
        processed_image = self._preprocess_image(image_path)
        
        # 2. 转换为base64
        base64_image = self._image_to_base64(processed_image)
        
        # 3. 构建提示词
        prompt = """请识别图片中的垃圾类别，按以下JSON格式返回：
{
    "category": "可回收物/有害垃圾/厨余垃圾/其他垃圾",
    "confidence": 0.95,
    "explanation": "分类理由说明",
    "suggestions": ["处理建议1", "处理建议2"]
}"""
        
        # 4. 调用多模态模型
        messages = [
            {
                "role": "user",
                "content": [
                    {"type": "text", "text": prompt},
                    {"type": "image_url", "image_url": {"url": f"data:image/jpeg;base64,{base64_image}"}}
                ]
            }
        ]
        
        response = self.client.chat_completion(messages)
        return self._parse_response(response)
        
    def _preprocess_image(self, image_path, max_size=1024):
        """图片预处理：压缩、格式转换"""
        image = Image.open(image_path)
        
        # 等比缩放
        if max(image.size) > max_size:
            ratio = max_size / max(image.size)
            new_size = (int(image.size[0] * ratio), int(image.size[1] * ratio))
            image = image.resize(new_size, Image.Resampling.LANCZOS)
            
        # 转换为RGB
        if image.mode != 'RGB':
            image = image.convert('RGB')
            
        return image
        
    def _image_to_base64(self, image):
        import io
        buffer = io.BytesIO()
        image.save(buffer, format='JPEG', quality=85)
        return base64.b64encode(buffer.getvalue()).decode()

错误处理与降级策略

AI服务稳定性保障：
  • 超时重试：设置合理的超时时间，失败自动重试
  • 降级方案：主服务不可用时切换到备用模型
  • 限流保护：防止突发流量导致服务不可用
  • 缓存机制：对常见查询结果进行缓存
class AIClientWithFallback:
    def __init__(self, primary_client, fallback_client):
        self.primary = primary_client
        self.fallback = fallback_client
        self.retry_count = 3
        self.timeout = 30
        
    def chat(self, messages):
        """带降级策略的对话"""
        for attempt in range(self.retry_count):
            try:
                return self.primary.chat_completion(
                    messages, 
                    timeout=self.timeout
                )
            except TimeoutError:
                if attempt < self.retry_count - 1:
                    time.sleep(2 ** attempt)  # 指数退避
                    continue
                    
        # 主服务失败，使用降级方案
        print("Primary service failed, using fallback...")
        return self.fallback.chat_completion(messages)
