# 🎨 高俊杰的个人博客

一个现代化、美观的个人博客系统，支持Markdown自动转换和网页端文章管理。

## ✨ 特色功能

### 🎭 现代化动画效果
- **渐变背景动画**：动态渐变色彩流动效果
- **卡片悬浮动画**：鼠标悬停时的3D提升效果
- **滚动显示动画**：元素滚动进入时的淡入动画
- **按钮波纹效果**：点击按钮时的水波纹动画
- **粒子背景效果**：可选的粒子动画背景

### 🎨 精美视觉设计
- **渐变配色方案**：紫色到粉色的现代渐变
- **响应式布局**：完美适配各种设备
- **流畅交互**：平滑的过渡动画
- **精美排版**：舒适的阅读体验

### 💡 便捷文章管理
- **网页端编辑器**：直接在浏览器中编写文章
- **Markdown支持**：自动转换Markdown为HTML
- **实时预览**：即时查看文章效果
- **标签管理**：轻松添加和管理文章标签

### 🚀 其他特性
- **代码高亮**：支持多种编程语言语法高亮
- **阅读时间**：自动计算文章阅读时间
- **分类筛选**：按分类查看文章
- **搜索功能**：快速搜索文章内容

## 🚀 快速开始

### 方法一：使用Web服务器（推荐）

1. **启动服务器**
```bash
python server.py
```

2. **访问博客**
- 博客首页：http://localhost:5000
- 文章管理：http://localhost:5000/admin.html

3. **添加文章**
- 打开文章管理页面
- 填写文章信息
- 使用Markdown编写内容
- 点击保存即可

### 方法二：静态文件方式

1. **构建博客**
```bash
python build.py
```

2. **查看博客**
打开 `dist/index.html` 文件

## 📝 文章格式

Markdown文件需要包含frontmatter：

```markdown
---
title: 文章标题
date: 2026-03-06
category: 技术
tags: [标签1, 标签2]
excerpt: 文章摘要
cover: https://via.placeholder.com/800x400
author: 高俊杰
---

## 文章正文

这里是文章内容...
```

## 🎯 使用指南

### 添加新文章

**方法一：网页端添加（推荐）**
1. 访问 http://localhost:5000/admin.html
2. 填写文章信息
3. 使用Markdown编写内容
4. 点击保存

**方法二：命令行工具**
```bash
python new-post.py
```

**方法三：手动创建**
在 `posts` 目录创建 `.md` 文件

### 管理文章

- **编辑文章**：在文章管理页面点击编辑按钮
- **删除文章**：在文章管理页面点击删除按钮
- **查看文章**：在博客首页或列表页查看

## 🎨 动画效果展示

### 渐变背景
主页英雄区域使用动态渐变背景，颜色在紫色和粉色之间流动。

### 卡片动画
文章卡片在鼠标悬停时会：

- 放大到1.02倍
- 显示顶部渐变边框
- 增强阴影效果

### 按钮效果
按钮点击时会产生水波纹扩散效果，悬停时会向上浮动。

页面元素在滚动进入视口时会淡入显示，提升用户体验。

## 📖 开发说明

### 自定义样式
编辑 `css/style.css` 中的CSS变量：
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
}
```

### 添加新动画
在 `css/animations.css` 中添加新的动画关键帧和应用类。

### 扩展功能
- 修改 `server.py` 添加新的API端点
- 编辑 `js/admin.js` 扩展管理功能
- 更新模板文件改变页面结构
