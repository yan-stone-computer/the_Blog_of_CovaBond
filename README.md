# 🌃 高俊杰的赛博朋克博客

一个极简的赛博朋克风格个人博客系统。

## 🚀 快速开始

### 启动服务器
```bash
python server.py
```

或双击 `start.bat`

### 访问地址
- 🌐 **博客首页**：http://localhost:5000
- 📝 **博客列表**：http://localhost:5000/blog.html
- ⚙️ **文章管理**：http://localhost:5000/admin.html

## 📁 项目结构

```
PersonalBlog/
├── index.html          # 首页
├── blog.html           # 博客列表页
├── admin.html          # 文章管理页
├── css/
│   ├── style.css       # 主样式
│   └── animations.css  # 动画效果
├── js/
│   ├── main.js         # 主要功能
│   ├── admin.js        # 管理功能
│   └── ...
├── posts/              # Markdown文章
├── dist/               # 构建输出
├── server.py           # Web服务器
├── build.py            # 构建脚本
└── start.bat           # 启动脚本
```

## 🎨 赛博朋克特色

- 🌈 **霓虹配色**：青色、品红、黄色霓虹
- ⚡ **动画效果**：霓虹闪烁、故障艺术、扫描线
