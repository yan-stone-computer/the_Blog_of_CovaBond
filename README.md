# 🌃 高俊杰的博客

一个极简的赛博朋克风格个人博客系统，支持授权验证和图片上传。

## 🚀 快速开始

### 启动服务器
```bash
python server.py
```
或双击 `start.bat`

### 访问地址
- 🌐 **博客首页**：http://localhost:5000
- 📝 **博客列表**：http://localhost:5000/blog.html

## 📝 使用说明

### 添加文章
1. 访问 http://localhost:5000
2. 点击页面中央的 **新建文章** 按钮
3. 输入授权码：`gaojunjie2026`
4. 填写文章信息
5. 点击"上传"按钮上传封面图片
6. 使用Markdown编写内容
7. 点击"💾 保存"

### 图片上传
- 支持格式：PNG, JPG, JPEG, GIF, WEBP
- 图片保存在 `dist/uploads/` 目录
- 自动生成唯一文件名

### 头像设置
- 将您的头像文件命名为 `img.png`
- 放在项目根目录
- 自动显示在"关于我"区域
