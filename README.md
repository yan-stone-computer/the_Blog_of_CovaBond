# 🌃 高俊杰的赛博朋克博客

一个暗黑科技风格的个人博客系统，融合赛博朋克美学与现代Web技术。

## 🎨 赛博朋克视觉特色

### 🌈 霓虹配色
- **主色调**：青色霓虹 (#00ffff)
- **辅助色**：品红霓虹 (#ff00ff)
- **强调色**：黄色霓虹 (#ffff00)
- **暗黑背景**：深邃的暗色调 (#0a0a0f)

### ⚡ 动画效果
- **霓虹闪烁**：标题文字的霓虹灯闪烁效果
- **故障艺术**：Glitch故障效果
- **扫描线**：CRT显示器扫描线动画
- **数据流**：流动的数据流背景
- **脉冲发光**：按钮和卡片的脉冲发光
- **全息效果**：3D全息投影效果

### 🖥️ 视觉元素
- **扫描线滤镜**：模拟CRT显示器效果
- **霓虹边框**：发光的渐变边框
- **暗黑卡片**：半透明暗黑卡片设计
- **数据网格**：背景数据流网格

## 🚀 快速开始

### 启动服务器
```bash
python server.py
```

或双击 `start.bat` 一键启动

### 访问地址
- 🌐 **博客首页**：http://localhost:5000
- 📝 **文章管理**：http://localhost:5000/admin.html

## 📁 项目结构

```
PersonalBlog/
├── posts/              # Markdown文章
├── dist/               # 构建输出
├── css/
│   ├── style.css       # 主样式（赛博朋克配色）
│   └── animations.css  # 动画效果
├── js/
│   ├── main.js         # 主要功能
│   └── admin.js        # 管理页面
├── server.py           # Web服务器
├── build.py            # 构建脚本
└── start.bat           # 启动脚本
```

## 🎯 使用指南

### 添加文章
1. 访问 http://localhost:5000/admin.html
2. 填写文章信息
3. 使用Markdown编写内容
4. 点击保存

### 文章格式
```markdown

date: 2026-03-06
category: 技术

excerpt: 文章摘要
cover: https://example.com/image.jpg
author: 高俊杰
---

## 文章内容

使用Markdown编写...
```

## 🎨 自定义配色

编辑 `css/style.css` 中的CSS变量：

```css
:root {
    --primary-color: #00ffff;    /* 青色霓虹 */
    --secondary-color: #ff00ff;  /* 品红霓虹 */
    --accent-color: #ffff00;     /* 黄色霓虹 */
    --bg-dark: #0a0a0f;          /* 暗黑背景 */
}
```
