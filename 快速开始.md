# 🚀 快速开始指南

## 📝 如何添加新文章

### 方法一：使用文章创建工具（推荐）

运行文章创建工具：

```bash
python new-post.py
```

按照提示输入文章信息：
1. 选择"1. 创建新文章"
2. 输入文章标题
3. 选择分类（技术/生活/学习/其他）
4. 输入标签（用逗号分隔）
5. 输入文章摘要
6. 输入封面图片URL（可选）
7. 输入作者名称（可选）
8. 输入文章内容（输入完成后在新行输入 `END` 结束）

### 方法二：手动创建Markdown文件

在 `posts` 目录下创建 `.md` 文件，格式如下：

```markdown
---
title: 文章标题
date: 2026-03-06
category: 技术
tags: [标签1, 标签2]
excerpt: 文章摘要
cover: https://via.placeholder.com/800x400
author: 博主
---

## 文章正文

这里是文章内容...
```

## 🔨 构建博客

创建文章后，运行构建命令：

```bash
python build.py
```

## 👀 查看效果

打开 `dist/index.html` 文件查看博客效果。

## 📋 完整流程

1. **创建文章**：`python new-post.py`
2. **构建博客**：`python build.py`
3. **查看效果**：打开 `dist/index.html`

## 💡 提示

- 文章会自动按修改时间排序
- 支持标准Markdown语法
- 代码块会自动高亮
- 阅读时间会自动计算

## 🎯 示例

让我们创建第一篇文章：

```bash

```

然后按提示操作即可！
